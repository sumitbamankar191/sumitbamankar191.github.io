Welcome to my personal portfolio repository! This project is a modern, high-performance website designed to showcase my dual expertise as a Graphic Designer and a Student Developer. It features smooth animations, a responsive interface, and detailed system architecture diagrams.
🔗 Live Demo: View My Portfolio (https://sumitbamankar191.github.io/)
🚀 Key Features
Dynamic UI/UX: Built with a "mobile-first" approach using Bootstrap 4.
Engaging Animations: Integrated AOS (Animate on Scroll) and Stellar.js for a professional parallax feel.
Interactive Components: Typing effects via Typed.js and smooth project sliders using Owl Carousel.
System Analysis: Includes built-in Data Flow Diagrams (DFD) and Entity Relationship (ER) diagrams to demonstrate technical planning.
Contact Ready: Fully integrated with Google Forms for easy communication.
🛠️ Tech Stack
Category	Tools & Technologies
Frontend	HTML5, SCSS (Compiled to CSS), Vanilla JavaScript, jQuery
Frameworks	Bootstrap 4
Libraries	AOS, Owl Carousel, Waypoints, Magnific Popup, Typed.js
Diagrams	Mermaid.js (Architecture visualization)
Design	Flaticon, Icomoon, Google Fonts (Poppins)
📊 System Architecture
Unlike standard portfolios, this project includes a deep dive into its own logic:
DFD (Level 0 & 1): Visualizes how data flows from the user interface to the DOM and external APIs.
ER Diagram: Models the structure of my projects, skills, and certifications.
Access: View these via the dfd-gallery.html and er-gallery.html files included in the project.
